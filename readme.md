# Movie List 
A movie browsing website which can look into more details, e.g. plot summary, release date..

## Features
- Render Moives info as a Card List from API
- Can search movie by typing keyword of movie title
- Add and store those movies you like
- memorize all the chosen moveis in favorite site

### Searching bar
Type in the keyword or substring in searching bar

### Favorite list
Use "+" button to store the movie,
switch to favorite list to check out those movies.
Can remove them by "X" button.
